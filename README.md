🚀 K. Jeevan Kumar – Portfolio

A premium, interactive personal portfolio website built with React, TypeScript, and TailwindCSS.
This portfolio highlights projects such as LeakGuard AI and Currency Recognition Systems, with smooth animations, dark mode support, and a clean UI designed for AI/ML roles.

🔗 Live Demo:
https://kjeevankumar.github.io/kjeevankumar.g1/

✨ Features

⚡ Modern Tech Stack – Built with Vite, React 18, and TypeScript

🎨 Beautiful UI – Styled using TailwindCSS and Shadcn UI components

🎬 Smooth Animations – Powered by Framer Motion and custom CSS effects
(Glassmorphism, levitate effects)

🌙 Dark Mode Support – Custom theme toggle for light/dark modes

🌍 GitHub Pages Ready – Configured for sub-folder hosting using HashRouter and gh-pages

🛠 Tech Stack

🚀 Getting Started

Follow the steps below to run the project locally.

1️⃣ Prerequisites

Make sure you have the following installed:

Node.js (v16 or higher recommended)

Git

2️⃣ Clone the Repository

Open your terminal and run:

git clone https://github.com/kjeevankumar/kjeevankumar.g1.git
3️⃣ Navigate to the Project Folder
cd kjeevankumar.g1
4️⃣ Install Dependencies
npm install
5️⃣ Run the Development Server
npm run dev

The project will start locally at:

http://localhost:8080/
🌍 Deployment

This project is configured for easy deployment to GitHub Pages.

Step 1 — Commit your changes
git add .
git commit -m "Update portfolio"
git push
Step 2 — Deploy to GitHub Pages

Run:

npm run deploy

This will:

Build the production files into the dist/ folder

Push the build to the gh-pages branch

Step 3 — Enable GitHub Pages

Go to your repository:

Settings → Pages

Set the source to:

Branch: gh-pages
Folder: / (root)

Your portfolio will be live at:

https://kjeevankumar.github.io/kjeevankumar.g1/
